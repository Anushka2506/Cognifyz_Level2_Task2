# Level 2 Task 2: Responsive Design & Hamburger Menu

## Project Overview
This project demonstrates how to convert a static webpage into a fully responsive design using **Pure CSS Media Queries**. No external frameworks like Bootstrap were used for this specific task to demonstrate a deep understanding of CSS layout logic.

## Key Features
- **Mobile-Friendly Navigation:** A horizontal menu that collapses into a functional "Hamburger Menu" on screens smaller than 768px.
- **Pure CSS Toggle:** Implemented the "Checkbox Hack" to toggle the mobile menu without using JavaScript.
- **Responsive Grid:** A flexible layout where content boxes stack vertically on mobile devices.
- **Smooth Animations:** CSS transitions for a premium user experience.

## Technologies Used
- HTML5
- CSS3 (Flexbox & Media Queries)
- Font Awesome (for icons)
